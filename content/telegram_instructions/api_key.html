2. API-ключ Боту нужен API-ключ для выставления ордеров. Как создать: 1. Зайдите в созданный Субаккаунт. 2. Перейдите в раздел API → Создать новый ключ. 3. Выберите API-ключи, созданные системой. 4. В настройках ключа укажите: • Чтение и запись • Нет ограничений по IP • Разрешения: Единый торговый аккаунт (4 галочки: Ордера/Позиции, Торговля USDC, Торговля) • ❌ ВАЖНО: Убедитесь, что галочки Активы (Обмен, Кошелек) СНЯТЫ. Бот не должен иметь доступа к выводу денег! 5. Нажмите "Отправить". 6. Скопируйте API Key и API Secret (Секретный ключ покажут только один раз). Ввод в боте: 1. В главном меню бота нажмите Настройки → Указать API ключи. 2. Отправьте боту ключи одной строкой через пробел: API_KEY API_SECRET
