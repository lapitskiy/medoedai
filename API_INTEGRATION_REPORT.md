## Как использовать

### 1. Настройка API ключей
Установите переменные окружения:

```bash
export BYBIT_API_KEY="your_actual_api_key_here"
export BYBIT_SECRET_KEY="your_actual_secret_key_here"
```

### 2. Проверка настройки
При запуске приложения вы увидите:
- ✅ "API ключи Bybit настроены" - если ключи установлены
- ⚠️ "API ключи Bybit не настроены" - если ключи отсутствуют

### 3. Использование в торговле
API ключи автоматически передаются в:
- Функцию `trade_step()` для выполнения торговых операций
- Контейнер `trading_agent` через переменные окружения
